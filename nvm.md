### nvm 常用操作

https://blog.csdn.net/ganyingxie123456/article/details/132188576

// 安装指定版本的node
nvm install node版本号（如：nvm install 14.16.0）

// 使用指定版本的node
nvm use node版本号

// 使用最新版本的node
nvm use node

// 卸载指定版本的node
nvm uninstall node版本号

// 设置node别名
nvm alias current-version node版本号

// 取消别名
nvm unalias current-version
